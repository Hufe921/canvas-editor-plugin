# Installation

## Requirements

- Node.js >= 14
- canvas-editor >= 0.9.0

## Install Plugins

Each plugin is an independent npm package and can be installed separately:

### Using npm

```bash
npm install @hufe921/canvas-editor-plugin-barcode1d
npm install @hufe921/canvas-editor-plugin-barcode2d
npm install @hufe921/canvas-editor-plugin-codeblock
```

### Using yarn

```bash
yarn add @hufe921/canvas-editor-plugin-barcode1d
yarn add @hufe921/canvas-editor-plugin-barcode2d
yarn add @hufe921/canvas-editor-plugin-codeblock
```

### Using pnpm

```bash
pnpm add @hufe921/canvas-editor-plugin-barcode1d
pnpm add @hufe921/canvas-editor-plugin-barcode2d
pnpm add @hufe921/canvas-editor-plugin-codeblock
```

## Install canvas-editor

Before using plugins, you need to install canvas-editor:

```bash
npm install @hufe921/canvas-editor
```
