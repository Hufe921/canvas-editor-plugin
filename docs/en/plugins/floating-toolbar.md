# Floating Toolbar

Floating toolbar plugin that displays quick action toolbar when text is selected.

## Installation

```bash
npm install @hufe921/canvas-editor-plugin-floating-toolbar
```

## Usage

```javascript
import Editor from '@hufe921/canvas-editor'
import floatingToolbarPlugin from '@hufe921/canvas-editor-plugin-floating-toolbar'

const instance = new Editor()
instance.use(floatingToolbarPlugin)
```

## Description

This plugin requires no additional configuration. After installation, the floating toolbar will automatically appear when text is selected, providing quick formatting operations.
